# Lista 02

## Aluno: Pedro Henrique Tomaz Romão

### Objetivo: Compreender e aplicar a estrutura display flex e display grid, bem como seus atributos align, justify e template.

---  

Exercicio 1 - Utilização de uma tag header para o cabeçalho e display flex para formatação dos espaços e alinhamentos.

---  

Exercicio 2 - Utilização de uma tag main para o corpo e display flex para quebra e alinhamento de linha. Uso de column-gap com calc para manter no maximo 4 cards por linha.

---  

Exercicio 3 - Utilização de uma tag main para o corpo e display grid para formatação das imagens. Grid-template-columns para definir a divisão e grid-column para definir a ocupação do espaço

---  

Exercicio 4 - Uso de diversas tags semanticas para a estrutura do projeto e uso de grid-template-areas e grid-area para definição da ocupação dos espaços. Uso também de display flex para organização interna das tags e do conteúdo.

---  

Exercicio 5 - Uso de diversas tags semanticas para a estrutura do projeto e uso de grid-template-areas e grid-area para definição da ocupação dos espaços, em especial grid-template-rows auto para ocupação minima do header e footer. Uso também de display flex para organização interna das tags e do conteúdo, uso principal de flex-direction, margin auto para deslocar elementos, gap e padding para espaçamentos.
